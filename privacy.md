# Privacy Policy (Italiano)

## Ultimo aggiornamento: 14 Dicembre 2025

La presente Privacy Policy descrive le modalità di gestione dell'applicazione **Doctor Love** (di seguito "l'Applicazione") in riferimento al trattamento dei dati personali degli utenti.

## 1. Titolare del Trattamento

L'applicazione è sviluppata come progetto indipendente. Per qualsiasi informazione, è possibile contattare gli sviluppatori.

## 2. Tipologia di Dati Raccolti

L'Applicazione ha accesso solo ed esclusivamente alle immagini (screenshot) che l'utente seleziona volontariamente o cattura tramite la funzione "Scanner Live".
**Nessun dato personale** (nome, email, numero di telefono) viene richiesto o raccolto dall'Applicazione.

## 3. Modalità del Trattamento (Local First)

Doctor Love è progettata con un approccio **Privacy-First**:

* **Analisi Locale:** L'elaborazione delle immagini avviene, ove possibile, utilizzando le API sicure di Google Gemini.
* **Nessuna Archiviazione Remota:** Le immagini analizzate **NON** vengono salvate su nessun server esterno di nostra proprietà. Vengono inviate esclusivamente ai server di Google per l'analisi immediata e distrutte subito dopo l'elaborazione.
* **Dati nel Dispositivo:** I contatori delle analisi, le preferenze (tema, lingua) e l'eventuale API Key personale sono salvati **esclusivamente nella memoria sicura del dispositivo** dell'utente.

## 4. API Key Personale (BYOK - Bring Your Own Key)

Se l'utente sceglie di inserire la propria API Key di Google Gemini:

* La chiave viene salvata nel **Secure Storage** crittografato del sistema operativo (Android Keystore).
* La chiave viene utilizzata **solo** per effettuare le richieste di analisi verso Google.
* La chiave **non viene mai condivisa** con terze parti né inviata ai nostri server.

## 5. Finalità del Trattamento

I dati (screenshot) sono trattati esclusivamente per fornire il servizio di "Analisi della chat" richiesto dall'utente.

## 6. Diritti dell'Utente

L'utente può in qualsiasi momento:

* Cancellare tutti i dati salvati nell'app tramite il pulsante "Elimina Dati" nelle Impostazioni.
* Disinstallare l'app, rimuovendo così ogni traccia locale.

## Privacy Policy (English)

### Last Updated: December 14, 2025

This Privacy Policy describes how **Doctor Love** (hereinafter "the Application") manages user personal data.

## 1. Data Controller

The application is developed as an independent project. For any information, you can contact the developers.

## 2. Types of Data Collected

The Application has access exclusively to the images (screenshots) that the user voluntarily selects or captures via the "Live Scanner" feature.
**No personal data** (name, email, phone number) is requested or collected by the Application.

## 3. Data Processing Methods (Local First)

Doctor Love is designed with a **Privacy-First** approach:

* **Local Processing:** Image processing is performed using secure Google Gemini APIs.
* **No Remote Storage:** Analyzed images are **NOT** saved on any external servers owned by us. They are sent exclusively to Google's servers for immediate analysis and destroyed immediately after processing.
* **On-Device Data:** Analysis counters, preferences (theme, language), and any personal API Key are saved **exclusively in the device's secure memory**.

## 4. Personal API Key (BYOK - Bring Your Own Key)

If the user chooses to enter their own Google Gemini API Key:

* The key is saved in the operating system's encrypted **Secure Storage** (Android Keystore).
* The key is used **only** to make analysis requests to Google.
* The key is **never shared** with third parties nor sent to our servers.

## 5. Purpose of Processing

Data (screenshots) are processed exclusively to provide the "Chat Analysis" service requested by the user.

## 6. User Rights

The user can at any time:

* Delete all data saved in the app via the "Delete Data" button in Settings.
* Uninstall the app, removing all local traces.
