# Privacy Policy – Doctor Love

Doctor Love analizza screenshot di chat per valutare il livello di interesse.

- **Non salviamo** nessun dato (immagini, chat, messaggi).  
- Tutto avviene sul tuo telefono o tramite API terze (Gemini, Pixtral, Qwen).  
- Se inserisci la tua API Key Google, resta **solo sul tuo dispositivo** (non la vediamo né la salviamo).  
- Nessun account richiesto, nessun dato personale raccolto.  
- App utilizzabile da tutti (nessun limite di età).
